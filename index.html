Updates

  • 26th September 2013: Initial version of tool released.
  • Full history of bugfixes is not so interesting...
  • 30th June 2017: Added support for board-style projects.
  • 20th August 2017: Added support for custom fields. Made possible by generous support from Gauge°
  • 10th May 2018: Added simple usage instructions - thank you to everyone who's suggested this, it should have been there from the start!
  • 15th October 2018: Several bugs fixed preventing tasks and tags from being created.
  • 20th March 2019: Organise Asana is deprecated and replaced with Ditto.
  • 11th February 2020: Asana numeric IDs switched off in API. Organise Asana will no longer function, and you should consider using Ditto instead.
